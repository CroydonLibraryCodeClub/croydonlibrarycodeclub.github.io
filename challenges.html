Weekly code challenges

Each week we set coding challenges for our club members. Click on Newsletters to see our current challenge.

What can I use?

You can complete these challenges using Scratch, HTML, Python, or other programming language as long as you are able to share your results with us.

When is the deadline?

You must submit your entry by the following Thursday

How do I submit?

Please get a parent to send your first name, age, and project link to paul.sinnett@gmail.com

If you are using Scratch, make sure you have shared your project and then send us the link.

If you are using HTML or Python with Trinket, publish your page and send us the link.

If you are using another language or system, you will need to find your own way to share your code with us.
