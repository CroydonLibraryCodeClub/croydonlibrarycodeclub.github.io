3D

Itʼs time to enter the third dimension. This week we will try making things in 3D!

Getting Started

Unfortunately, Scratch doesn't support 3D, so we will be using BlocksCAD or Blender for these projects. The project guides linked below include links for BlocksCAD and Blender.

BlocksCAD is a bit like Scratch. You can sign up on their website. And if you have a 3D printer you can even print out your models in 3D.

Blender is a 3D animation program that you will need to download and install.

You can design a 3D coder keyring with this project guide. Or click here for a video tutorial.

The Party Monkey project shows you how to build a party hat for a monkey in Blender. Here is a video walk-through of that project.

Here is a video showing how to build a 3D bug using the 3D bug project.

Or maybe you can use these ideas as inspiration for making your own 3D models.
