Something new to try

If you want to try Python to make your code club theme projects, why not give pgzero a try.

pgzero is a Python library that makes it easier to turn your Scratch projects into Python projects.

If you have Python installed, you can install pgzero by entering the following command in your Windows Command prompt, or Mac or Linux Terminal:

					
pip install pgzero
